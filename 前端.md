# 17.background

```css
1.参数是方位名词
	如果指定的两个值都是方位名词，则两个值前后顺序无关，比如left top和top left效果一致
	如果只指定了一个方位名词，另一个值省略，则第二个值默认居中对齐

2.参数是精确单位
	如果参数值是精确坐标，那么第一个肯定是x坐标，第二个一定是y坐标
	如果只指定一个数值，那该数值一定是x坐标，另一个默认垂直居中

3.参数是混合单位
	如果指定的两个值是精确单位和方位名词混合使用，则第一个值是x坐标，第二个值是y坐标

background-color: transparent; (透明) // 默认值

background-image: url();	
background-repeat: no-repeat; // 图片不平铺
background-repeat: repeat-x;  // 横向平铺
background-repeat: repeat-y;	// 纵向平铺

// 如果指定的两个值都是方位名词，则两个值前后顺序无关，比如left top和top left效果一致
background-position: top | center | bottom = x  left | center| right = y // 方位名词
// 如果只指定一个数值，那该数值一定是x坐标，另一个默认垂直居中
background-position: 20px 50px // 精准单位
```

# 29.z-index（定位叠放顺序）

```css

数值可以是正整数、负整数或O,默认是auto，数值越大，盒子越靠上
如果属性值相同，则按照书写顺序，后来居上
/* 只有定位的盒子才有z-index属性 (标准流和浮动没有该属性)*/
.content {
  position: relative;
  left: 500px;
  width: 300px;
  height: 300px;
  background: saddlebrown;
}
.text {
  position: absolute; 
  width: 100px; 
  height: 100px;
  z-index: 10;
  background: red;
}
.s { 
  position: absolute;
  width: 100px;
  height: 100px; 
  z-index: 1;
  background: seashell;
}

<div class="content"> 
  <div class="text"></div> 
  <div class="s"></div>
</div>
```
